Module for my function print_square

General properties:

prints rows and columns of the symbol # - the length of the rows and columns is equal to size
:

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(1)
    #

:

    >>> print_square(5)
    #####
    #####
    #####
    #####
    #####

:

    >>> print_square(10)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########

if the size is nil, there is no square

:

    >>> print_square(0)

Potential errors:

Size should be an int otherwise typerror

:

    >>> print_square("Omar")
    Traceback (most recent call last):
    TypeError: size must be an integer

:

    >>> print_square(3.14)
    Traceback (most recent call last):
    TypeError: size must be an integer

:

    >>> print_square(None)
    Traceback (most recent call last):
    TypeError: size must be an integer

If ``size`` is negative, a valueerror is raised.

:

    >>> print_square(-2)
    Traceback (most recent call last):
    ValueError: size must be >= 0

type checks will have priority over value checks. ie value is checked only and only if the type is accurate.

:

    >>> print_square(-3.14)
    Traceback (most recent call last):
    TypeError: size must be an integer

size must be provided. Might add support for standard size later, maybe set it to 1.
