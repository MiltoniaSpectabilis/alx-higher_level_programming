Module for my function add_integer

General properties:

add_integer() is equivalent to using the + operator and only works for int and float.

:

    >>> add_integer = __import__('0-add_integer').add_integer
    >>> add_integer(1, 3)
    4

:

    >>> add_integer(3, -3)
    0


casting performed prior to addition

:

    >>> add_integer(2.7, 0.2)
    2

The function supports floats.

:

    >>> add_integer(1.0, 3.0)
    4

:

    >>> add_integer(-2.9, -0.2)
    -2



The second argument is by default 98, if not specified.

:

    >>> add_integer(2)
    100


Type mixing supported.

:

    >>> add_integer(2.3, -3)
    -1
